About | github-stats-pages

This software was envisioned as part of a way to preserve traffic statistics for any public GitHub repository. Traffic statistics is one way to measure the impact of open-source software. The idea of this project came to fruition when I was working on an open-source software that I built for astronomers where it received a fair amount of use. Because GitHub only stores that record for up to two weeks, this poses a problem for measuring impact over time. While there were some solutions available on GitHub, I wanted something that would generate it for multiple repositories and to publish static site content. In principle these pages can be deployed in a number of ways, including GitHub Pages. I hope the project can expand to include other statistics so that it is a one-stop website for all GitHub statistics.
